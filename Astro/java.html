关于使用 Java 客户端进服的详细公告 ​

如果您对基岩版的游戏优化不满意，并希望使用 Java 客户端进行游戏，但没有电脑设备，可以参考以下详细指南：

推荐使用 FCL 启动器 ​

FCL 启动器 是一个优秀的选择，可以帮助您在移动设备上顺利运行 Java 客户端。请按照以下步骤进行操作：

  • 下载 FCL 启动器：
    • 推荐下载链接: FCL 启动器
    • 官方网站: FCL 启动器官网

使用 Pojavlauncher ​

如果您更倾向于使用 Pojavlauncher 来运行 Java 客户端，以下是详细的下载和安装步骤：

  • Pojavlauncher：

    • 官方网站: Pojavlauncher 官网

  • 安卓设备下载：

    • Google Play 商店下载链接: Pojavlauncher 安卓版

  • 苹果 (iOS) 设备下载：

    1. 安装巨魔商店（TrollStore）：

      • 访问以下地址下载并安装 巨魔商店: TrollStore 下载

    2. 下载最新版 Pojavlauncher：

      • 访问以下地址下载 Pojavlauncher 的最新版本： Pojavlauncher iOS 最新版

    3. 安装 Pojavlauncher：

      • 找到下载的 .ipa 文件。
      • 点击文件旁边的“共享”图标。
      • 在可用的选项列表中选择 TrollStore，以完成安装。

如您在安装过程中遇到任何问题或需要额外帮助，请随时联系服务器支持团队。我们致力于为您提供最佳的游戏体验。
